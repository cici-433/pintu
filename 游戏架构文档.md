# 旧时光照相馆 · 游戏架构文档

## 1. 总体架构
- 技术栈：Cocos Creator（前端渲染与交互）+ 抖音小游戏平台导出
- 架构约束：纯本地实现，无服务端；不考虑用户手动更改系统时间导致的异常
- 分层设计：
  - UI层（场景与界面）：Main、Game、Gallery、弹窗
  - 业务逻辑层（Services）：EnergyManager、PuzzleBoard、Stopwatch
  - 数据层（LocalStorage）：体力状态、已收集图片ID列表

## 2. 场景与页面
- Main（首页）
  - 展示每日剩余次数与倒计时；小于5次显示“🎞️ ＋”
  - 入口：开始游戏、我的相册；设置按钮
- Game（游戏页）
  - 右上角次数与倒计时；“🎞️ ＋”可唤起体力弹窗
  - 状态栏：TIME/STEP；拼图棋盘；提示/重置；胜利弹窗
- Gallery（相册）
  - 照片墙：展示通关收集的图片；不显示体力/次数
- Modals（弹窗）
  - 设置、原图预览、体力不足、结算

## 3. 业务模块
- EnergyManager（体力管理） [energy.ts](file:///Users/chenxunlin/小程序/拼图/core/energy.ts)
  - 状态：remaining、date、lastRecoverAt
  - 能力：load/save/resetIfCrossDay/consumeOne/recoverOne/tick/getNextRecoverMs/getMax
  - 规则：每日初始5次；每消耗1次后每10分钟恢复1次至上限；跨天重置
- PuzzleBoard（拼图逻辑） [puzzle.ts](file:///Users/chenxunlin/小程序/拼图/core/puzzle.ts)
  - 状态：size、state、emptyIndex
  - 能力：neighbors/shuffle/canMove/move/isSolved/getState
  - 洗牌：从已解状态出发进行合法随机移动，保证可解
- Stopwatch（计时） [timer.ts](file:///Users/chenxunlin/小程序/拼图/core/timer.ts)
  - 能力：start/stop/reset/getElapsedMs/format(mm:ss)

## 4. 状态与数据
- 用户进度：collectedPhotos（number[]，本地存储）
- 当前关卡：currentLevelId（number）
- 体力状态：EnergyState（remaining/date/lastRecoverAt）

## 5. 事件流与交互
- 首页开始游戏
  1) 点击“开始游戏”→ 检查体力；体力>0则扣1并进入第1关；=0弹出体力弹窗
  2) 小于5次时显示“🎞️ ＋”，点击弹窗可恢复一次
- 游戏内链路
  1) 进入关卡→ 计时开始→ 点击相邻块移动→ 胜利判定
  2) 胜利→ 收集图片→ 下一张（同链路不再扣次）
- 恢复机制
  - tick 定时更新倒计时与自动恢复；弹窗同步显示预计恢复时间

## 6. 关卡与资源
- 关卡数量：100；尺寸递进：第1关3×3，第2关4×4，第3关5×5，第4关及后续6×6
- 资源：目标图片切图；棋盘格按 val 计算 bg-position；占位素材后续替换

## 7. 边界与错误策略
- 时间更改：不校验系统时间更改导致的异常
- 存储不可用：降级到内存状态（不持久化），提示用户功能受限
- 拼图渲染：状态驱动刷新，避免直接 DOM 位置错乱（在引擎中以节点重排实现）

## 8. 引擎适配与构建
- Cocos 场景建议
  - MainScene：EnergyManager 接口绑定，导航到 Game/Gallery
  - GameScene：PuzzleBoard+Stopwatch；UI状态栏与棋盘节点
  - GalleryScene：照片墙节点；点击查看大图
- 构建与导出
  - 使用 Cocos 的抖音小游戏导出模板；激励广告使用占位回调接口（后续接入）

## 9. 接口规范（TS）
- 导出入口 [index.ts](file:///Users/chenxunlin/小程序/拼图/core/index.ts)
  - import { PuzzleBoard, Stopwatch, EnergyManager } from './core'
- 使用示例
  - PuzzleBoard：`const b = new PuzzleBoard(size); b.shuffle(); b.move(index); b.isSolved();`
  - Stopwatch：`const sw = new Stopwatch(); sw.start(); sw.stop(); sw.format();`
  - EnergyManager：`const em = new EnergyManager(); em.consumeOne(); em.recoverOne(); em.tick();`

## 10. 验收与监控
- 功能验收：体力扣减/恢复一致性；倒计时显示；胜利判定正确；相册收集正确
- 指标记录：本地调试日志开关；不记录敏感信息
