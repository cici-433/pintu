# 游戏设计文档 (GDD) - 《旧时光照相馆》

## 1. 项目概述
| 项目名称 | 旧时光照相馆 (OldTimeStudio) |
| :--- | :--- |
| **游戏类型** | 益智 / 拼图 / 怀旧 |
| **核心玩法** | 滑动拼图 (华容道机制) |
| **美术风格** | 70-80年代复古风 |
| **开发引擎** | Cocos Creator (TypeScript) |
| **目标平台** | 抖音小游戏 |

## 2. 核心概念 (Core Concept)
玩家扮演一名“旧时光照相馆”的修复师，顾客带来一张张破损或错乱的泛黄老照片。玩家通过移动照片碎片（滑动拼图），将混乱的记忆重新归位，修复照片，还原那个年代的珍贵瞬间。

## 3. 游戏流程 (Game Loop)
1.  **进入游戏**：展示复古照相馆门面，点击“开始营业”。
2.  **选择关卡/相册**：选择一本相册（如“童年玩具”、“家庭合影”、“街头巷尾”）。
3.  **开始修复**：进入拼图界面，展示一张被打乱的老照片。
4.  **操作玩法**：点击空格旁边的碎片，将其移动到空格处，直到所有碎片归位。
5.  **成功结算**：照片修复完成，变成彩色或高清晰度，播放一段怀旧音效/文案。
6.  **奖励与解锁**：获得“胶卷”或“好评”，解锁下一张照片。

## 4. 详细玩法设计

### 4.1 核心机制：滑动拼图
-   **网格系统**：
    -   初级：3x3 网格 (8个碎片 + 1个空格)
    -   中级：4x4 网格 (15个碎片 + 1个空格)
    -   高级：5x5 网格 (24个碎片 + 1个空格)
-   **操作方式**：点击与空格相邻的方块，方块滑入空格。
-   **胜利条件**：所有碎片按照正确顺序排列（通常是左上角为1，右下角为空格）。

### 4.2 关卡设计 (Level Design)
游戏分为不同的“相册”章节，每个章节包含若干关卡。

*   **第一章：童年记忆 (3x3 简单)**
    *   Level 1-1: 铁皮青蛙
    *   Level 1-2: 玻璃弹珠
    *   Level 1-3: 小霸王游戏机
*   **第二章：生活物件 (4x4 中等)**
    *   Level 2-1: 二八大杠自行车
    *   Level 2-2: 老式缝纫机
    *   Level 2-3: 显像管电视机
*   **第三章：时代印记 (5x5 困难)**
    *   Level 3-1: 供销社柜台
    *   Level 3-2: 露天电影院
    *   Level 3-3: 春节联欢晚会

### 4.3 辅助功能
-   **原图预览**：按住按钮查看完整照片（有次数限制或冷却时间）。
-   **数字提示**：在碎片上显示数字编号（可开关），降低难度。
-   **重置**：重新打乱当前关卡。

## 5. 美术与UI设计 (Art & UI)
-   **色调**：泛黄、褐色、低饱和度，带有胶片颗粒感。
-   **字体**：衬线体、魏碑或手写体，模仿旧报纸或手写信件。

## 6. 音效设计 (Audio)
-   **BGM**：轻柔的口琴、手风琴旋律，带有留声机的底噪。
-   **点击音效**：清脆的木块碰撞声，或者相机快门声“咔嚓”。
-   **胜利音效**：一段欢快的复古旋律，类似老电视节目的过场音乐。

## 7. 商业化与社交 (Monetization & Social)
-   **广告点**：
    -   查看原图（激励视频）
    -   解锁新相册（激励视频）
    -   体力不足时补充（激励视频）
-   **分享点**：
    -   “我成功修复了这张[铁皮青蛙]照片，快来看看你的童年回忆！”
    -   生成一张带有复古滤镜的拍立得海报用于分享。
